Benefits of Mindful Habits

Improved Mental Clarity

Enhanced Mood

Regular mindful practices have been shown to reduce stress and improve emotional wellbeing.

About This Project

I created the Mindful Habits Tracker because I've been trying to build better daily habits to improve my own physical and mental well-being. This project is a way for me to apply what I've learned in web development while also designing a tool that helps me stay consistent with small, meaningful routines like drinking water, stretching, and getting enough sleep.

The website features a home page introducing mindful daily habits, a dynamic habit tracker with progress saving and user input, and a reference page with additional resources. All built with responsive design and interactive JavaScript to promote wellness and healthy routines.
